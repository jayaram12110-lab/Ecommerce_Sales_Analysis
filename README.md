# 🛒 E-Commerce Sales Analysis

## 📌 Project Overview
This project analyzes an e-commerce sales dataset using Python to identify sales trends, customer behavior, and business insights.

## 🎯 Objectives
- Analyze sales performance
- Identify top-selling products
- Compare sales by region
- Analyze profit and discounts
- Discover customer purchasing trends

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- VS Code

## 📂 Dataset
- Sample Superstore Dataset
- Sales, Profit, Discount, Category, Region, Customer information

## 📊 Analysis Performed
- Data Cleaning
- Missing Value Check
- Duplicate Value Check
- Sales by Category
- Profit by Region
- Monthly Sales Trend
- Top Customers
- Top Products
- Discount Analysis

## 📈 Key Insights
- Technology products generated the highest sales.
- Some regions achieved higher profits than others.
- Heavy discounts reduced overall profit.
- A small number of customers contributed a large portion of total sales.

## 🚀 Conclusion
This project demonstrates how exploratory data analysis (EDA) can be used to understand business performance and support data-driven decision making.
